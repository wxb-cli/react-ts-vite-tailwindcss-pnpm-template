# 项目模板

# 启动
node版本要求`18.12.0+`。   
使用 `pnpm` 按照项目依赖。

## 安装依赖
```
pnpm i
```

## 启动
```
pnpm run dev
```

## 打包
```
pnpm run build
```

# 技术栈
- Vite4.x
- TypeScript 5.x
- React 18.x
